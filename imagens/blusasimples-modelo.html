Para fazer com que uma linha (por exemplo, <hr> ou uma div com border) não ultrapasse o tamanho de uma imagem, você pode seguir estas opções em HTML + CSS:

✅ Opção 1: Usando um contêiner

Coloque a imagem e a linha dentro de uma div, e limite a largura do contêiner:

html
<div style="width: 300px; text-align: center;">
  <img src="sua-imagem.jpg" style="width: 100%;">
  <hr style="border: 1px solid #000; width: 100%;">
</div>

✅ Opção 2: Linha com o mesmo tamanho da imagem (sem div)

Se você não quiser usar um contêiner, pode dar à linha a mesma largura da imagem:

html
<img src="sua-imagem.jpg" style="width: 300px;">
<hr style="width: 300px; border: 1px solid #000;">

✅ Opção 3: Linha responsiva baseada na imagem

Se a imagem for responsiva (width: 100%), a linha também pode ser:

html
<img src="sua-imagem.jpg" style="width: 100%; max-width: 400px;">
<hr style="width: 100%; max-width: 400px; border: 1px solid #000;">

O z-index é uma propriedade CSS usada para controlar a ordem de empilhamento dos elementos — ou seja, qual elemento aparece por cima de outro.

📌 Regras básicas de z-index

  1. Só funciona em elementos posicionados (position: relative, absolute, fixed ou sticky).

  2. Maior valor = mais acima na pilha.

✅ Exemplo básico

html
<div class="caixa1">Caixa 1</div>
<div class="caixa2">Caixa 2</div>

css
.caixa1 {
  position: absolute;
  z-index: 1;
  background: red;
  width: 100px;
  height: 100px;
  top: 50px;
  left: 50px;
}

.caixa2 {
  position: absolute;
  z-index: 2; /* aparece por cima */
  background: blue;
  width: 100px;
  height: 100px;
  top: 70px;
  left: 70px;
}

💡 Dicas úteis

  • Se dois elementos têm o mesmo z-index, o que vem por último no HTML vai por cima.

  • z-index pode ser negativo, pra mandar um elemento pra trás.
